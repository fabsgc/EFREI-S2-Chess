## avancement du projet

Le projet comprend toutes les fonctionnalités minimales qu'il fallait coder : 

- affichage du jeu
- gestion des tours de jeux
- déplacement, coups spéciaux. 
- détection de fin de partie

La seule option qui a été terminée est la création d'une interface 2D. 
Une autre fonctionnalité qui a été rajoutée est la présence d'un menu.
